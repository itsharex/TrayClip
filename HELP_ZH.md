# TrayClip 帮助

## 概览
TrayClip 是一款纯本地剪贴板管理工具，围绕历史记录、分组、置顶、快速搜索和快速复制构建，同时提供主题切换、语言切换、备份恢复和可选的 AI 增强配置。

## 窗口操作
- 主窗口：可通过系统托盘图标、托盘菜单或全局快捷键打开。
- 窗口呼出位置：主窗口可按设置在屏幕中间打开，或跟随鼠标位置打开。
- 关闭行为：关闭主窗口默认会隐藏到托盘，可在设置中修改。

## 剪贴记录
- 自动采集：TrayClip 会自动记录剪贴板内容，包括纯文本、富文本、图片和文件路径。
- 复制回写：点击复制按钮，或双击记录内容区域，即可重新写回剪贴板。
- 右键菜单：右键点击记录可快速操作，JSON 内容支持格式化复制，包含链接时可直接打开。
- 置顶：置顶记录会被保留，不会参与自动清理。
- 删除：确认后可以永久删除记录。
- 搜索：搜索框可按内容、来源应用和文件路径过滤。
- 图片预览：图片记录会显示缩略图，文件路径中的图片也可以预览。
- 内容截断：超长文本会被截断，并标记为已截断。

## 分组管理
- 新建分组：点击分组栏右侧的 `+` 按钮，输入名称后按 `Enter`。
- 重命名分组：双击分组标签，输入新名称后按 `Enter` 确认，按 `Esc` 取消。
- 删除分组：右键点击分组标签并确认，组内记录会回到未分组状态。
- 筛选分组：点击分组标签筛选记录，点击 `全部` 查看所有记录。
- 移动记录到分组：在记录操作栏中选择目标分组即可。

## 键盘操作
- 使用 `↑` / `↓` 选择记录，按 `Enter` 复制当前记录。
- 双击记录内容区域可快速复制。
- 右键点击记录可打开上下文菜单。
- `Esc` 会先清空搜索；搜索为空时会隐藏窗口。
- 主窗口打开时会自动选中第一条可见记录，可直接按 `Enter` 复制。

## 设置说明
- 快捷键：可自定义主窗口快捷键，点击快捷键区域录入新组合，也可以恢复默认值。
- 历史记录保留条数：可设置 50 到 10000 之间的最大保留数量。
- 暂停采集：临时停止记录剪贴板内容。
- 快捷粘贴：仅 Windows 可用，从主窗口复制后会自动粘贴到之前的应用窗口。
- 链接提示：复制内容包含链接时显示提示窗口。
- 开机自动启动：系统启动时自动运行 TrayClip。
- 关闭行为：可选择隐藏到托盘、直接退出或每次询问。
- 窗口呼出位置：可选择屏幕中间或跟随鼠标。
- 主题和语言：可在标题栏切换亮色 / 暗色主题和界面语言。
- 数据备份：可导出包含记录、分组和设置的 ZIP 备份文件。
- 数据还原：导入之前导出的 ZIP 备份文件，会替换所有现有数据并重启应用。
- 清空历史：清空所有未置顶记录。
- AI 增强：可配置 API 地址、API Key、模型名称和 AI 翻译选项。
